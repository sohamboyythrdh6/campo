# campo
hey i am soham and i build this project for saftey reasons like it detected those spy cameras which recording us without our permeation and i build those project using  a microcontroller called raspberry pico and some photo diode and ir led  i use a custom pcb to build this project 
This project detects hidden cameras using IR light. The photodiode continuously reads the surrounding light and the system first checks the normal light level when it starts. Then it compares new readings with this normal value, and if the value suddenly increases, it means there is a strong reflection which may come from a camera lens. When this happens, the device triggers a buzzer and shows “DETECTED” on the OLED display.
